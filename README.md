# Nat's Recipe App

A recipe app built with Next.js, Tailwind CSS, and shadcn/ui. This is Nat's project - all UI decisions and features are hers to build.

## Tech Stack

- **Next.js** (App Router, TypeScript)
- **Tailwind CSS** v4
- **shadcn/ui** - Button, Card, and Input components pre-installed

## Getting Started

Install dependencies and run the dev server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the app.

The main page is `app/page.tsx` - edit it and the browser auto-updates.

```
app/           - Pages and layouts (App Router)
components/ui/ - shadcn/ui components (Button, Card, Input)
```
